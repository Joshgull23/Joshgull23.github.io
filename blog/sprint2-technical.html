Margin:

The margin is the space outside of the border and you can set the values around it using Top, Bottom, Left or Right to get the exact position you’d like the margin to be.

Border:

You can set the border to have styles or leave it transparent. It kind of really speaks for itself really.

Padding:

Padding is the direct distance from the elements content and helps distance the content away from seperate elements. It is very helpful and you can set any distance you’d like between the border of the box and the content. You can set padding (like the margin as well) to be Top, Bottom, Left or Right and generally use pixels (px).
